# Contribuir a WhatsApp Auto-Sender

¡Gracias por considerar contribuir a WhatsApp Auto-Sender!

## ¿Cómo Puedo Contribuir?

### Reportar Bugs

Antes de crear un reporte de bug, verifica los issues existentes. 
Al crear un reporte, incluye:

- **Título claro y descriptivo**
- **Pasos para reproducir el problema**
- **Comportamiento esperado vs actual**
- **Capturas de pantalla** si aplica
- **Detalles del entorno:**
  - Modelo de iPhone
  - Versión de iOS
  - Versión de a-Shell
  - Versión de Python

### Sugerir Mejoras

Al crear una sugerencia de mejora, incluye:

- **Descripción clara** de la funcionalidad
- **Por qué sería útil** para los usuarios
- **Posible enfoque de implementación**

### Pull Requests

1. Fork el repositorio y crea tu rama desde `main`
2. Si agregaste código, agrega pruebas
3. Asegúrate de que las pruebas pasen
4. Sigue el estilo PEP 8
5. Escribe un mensaje de commit claro
6. Actualiza la documentación si es necesario

## Configuración de Desarrollo
```bash
# 1. Clona tu fork
git clone https://github.com/tu-usuario/whatsapp-auto-sender-ios.git

# 2. Instala dependencias
pip install -r requirements.txt

# 3. Haz tus cambios

# 4. Prueba en iPhone con a-Shell

# 5. Envía PR
```

## Estilo de Código

- Sigue PEP 8
- Usa type hints cuando sea apropiado
- Agrega docstrings a funciones
- Mantén funciones pequeñas y enfocadas
- Comenta lógica compleja

## Mensajes de Commit

- Usa tiempo presente ("Agrega función" no "Agregó función")
- Usa modo imperativo
- Limita la primera línea a 72 caracteres
- Referencia issues cuando aplique

**Ejemplos:**
Agrega validación de números de teléfono
Verifica formato internacional
Agrega mensajes de error claros
Actualiza tests
Fixes #123
